Mesmer is a **cognitive hacking toolkit for LLMs**. It red-teams an LLM
target by running ReAct agents that pick profiling → planning →
execution moves from a tiered attack ladder — field-level payload
shapes *before* cognitive techniques — judges every attempt on a 1-10
rubric, and persists a per-target attack graph so every successive run
gets smarter.

## The premise

LLMs are minds, not filesystems. Static prompt banks — the shape most
red-teaming tools take — fail against modern targets because a frozen
attack can't adapt to a defense it hasn't seen. Mesmer treats attack
generation as a **search problem over cognitive space**: explore,
exploit, prune, remember.

Every module execution is recorded as an `AttackNode`. The executive
node is created at run start, manager modules attach under it, and
nested sub-module calls attach under the module that dispatched them.

Alongside that execution log, mesmer maintains a typed
[**belief graph**](/docs/concepts/belief-graph): weakness hypotheses
gain or lose confidence as the judge model extracts structured
evidence from each target reply, and a deterministic
[utility-ranked frontier](/docs/concepts/frontier-and-utility) names
the next move. The web UI's
[Belief Map view](/docs/recipes/reading-the-belief-map) renders this
state live.

## What you can do today

<ModuleGrid />

21 shipped modules: 4 managers (`system-prompt-extraction`,
`tool-extraction`, `indirect-prompt-injection`, `email-exfiltration-proof`), 1
profiler, 1 planner, 9 field-level techniques (T0/T1 — payload-shape
and structural injection, including function-calling probes), 6
cognitive / linguistic / psychological techniques (T2). All YAML,
all declarative, all picked up by `Registry.auto_discover()` on
startup. The depth-0 executive is synthesised by the runtime — never
authored.

---

Mesmer is open source, Python 3.10+, uses LiteLLM for all provider
dispatch, and ships as a CLI (`mesmer run`, `mesmer bench`,
`mesmer serve`) plus a FastAPI + Svelte 5 / D3 web UI. Cross-run state
lives outside the repo at `~/.mesmer/` so attack graphs survive across
sessions, and each bench trial writes a self-contained D3 HTML
mind-map you can open offline.
